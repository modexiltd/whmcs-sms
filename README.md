whmcs-sms
=========

Hemenposta WHMCS SMS Modül


WHMCS için açık kaynak kodlu, Ücretsiz SMS gönderme eklentisidir.


KURULUM
=========
 - Dosyaları WHMCS Klasörünüze ekleyin
 - Yönetim panelinizden *Menü->Ayarlar->Ek Modüller (Menu->Setup->Addon Modules) 
   sayasından HemenpostaSMS eklentisini etkinleştirin
 - Aktifleştirme işlemlerini gerçekleştirirken Yönetici için yetkileri etkinleştiriniz.
 - *Menü->Ayarlar->Müşteri Alanları (Menu->Setup->Custom Client Fields) sayfasından aşağıda belirtilen alanları oluşturun
